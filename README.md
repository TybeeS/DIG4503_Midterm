## Project Description

PollView is a website that houses a selection of user-submitted polls. Users can create polls on a variety of topics and submit these polls for others to vote on them. These polls can range from multiple choice to open-ended.

### Features:
- Easy creation of polls of various types, including multiple choice and open-ended responses.
- Vote on various user-submitted polls and view the results afterwards.
- View the results of various polls displayed on the website.
- Search/filter for polls by topic (e.g., politics, entertainment) for easy browsing.
- Track your own polls, voting history, and engagement analytics via user profile.

### Techstack: 
- Vanilla JavaScript
- Supabase for cloud database
- Supabase Auth or user authentication
- Tailwind CSS for styling
- Netlify for deployment
